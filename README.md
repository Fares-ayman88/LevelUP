# LevelUp Web (React)

This is the React web shell generated from the Flutter routes.

## Setup

1. Copy `.env.example` to `.env` and fill Firebase + PocketBase values.
2. Install dependencies and run dev server.

```
npm install
npm run dev
```

## Notes

- Routes are mapped in `src/routes.jsx`.
- Each screen is currently a placeholder in `src/pages/Placeholder.jsx`.
- Firebase auth/profile wiring lives in `src/state/auth.js`.
- PocketBase endpoint auto-detects a healthy base URL.
